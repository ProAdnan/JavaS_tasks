trim() : Removes spaces from both sides

trimStart() : Removes spaces from the beginning

trimEnd() : Removes spaces from the end

padStart() : Adds characters to the start

padEnd() : Adds characters to the end

charAt() : Returns the character at a given index

charCodeAt() : Returns ASCII/Unicode code of a character

split() : Splits the string into an array

length : Returns number of characters

slice() : Extracts part of a string (supports negative index)

substring() : Extracts part of a string (no negative index)

substr() : Extracts part using (start, length)

replace() : Replaces the first match

replaceAll() : Replaces all matches

toUpperCase() : Converts text to uppercase

toLowerCase() : Converts text to lowercase

concat() : Joins two or more strings

indexOf() : Returns first index of a match

lastIndexOf() : Returns last index of a match

search() : Searches string using regex

match() : Returns array of matches

matchAll() : Returns iterator of all matches

includes() : Checks if string contains a value

startsWith() : Checks if string starts with a value

endsWith() : Checks if string ends with a value
